# blog.mrinalini.dev

Personal blog and where I stash my notes. Built with vue, gridsome, grapql, vuex.
Link to blog --> https://blog.mrinalini.dev/

## Project setup

```
git clone https://github.com/mrinalini-m/blog.mrinalini.dev.git
```

```
cd blog.mrinalini.dev
```

```
yarn
```

or run `npm install`

```
gridsome develop
```
